# 🌵 Cactus Detection using Deep Learning

A deep learning project for cactus image classification using Convolutional Neural Networks (CNNs) and transfer learning with EfficientNetV2. The project compares different models and includes a simple application for image prediction.

---

## 📌 Project Overview

The goal of this project is to classify cactus images using deep learning techniques. Different models were trained and evaluated to determine the best-performing architecture.

The project includes:

- Data preprocessing and augmentation
- CNN model built from scratch
- Transfer Learning using EfficientNetV2
- Model evaluation and comparison
- Image prediction application
